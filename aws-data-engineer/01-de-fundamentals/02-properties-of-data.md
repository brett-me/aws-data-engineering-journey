# properties of data

## volume

- definition:
  - amount or size of data

- characteristics:
  - may range from gigabytes to petabytes or even more
  - impacts storage considerations and ability to analyze data

- examples:
  - social media platform processing terabytes of data daily from user posts
  - retailers collecting years worth of transaction data

## velocity

- definition:
  - speed at which new data is generated, collected and processed

- characteristics:
  - high velocity requires real-time or near-real-time processing capabilities
  - rapid ingestion and processing can be critical for some applications

- examples:
  - sensor data from IoT devices streaming readings every millisecond
  - high-frequency trading systems

## variety

- definition:
  - different types, structures and sources of data

- characteristics:
  - structured, semi-structured or unstructured
  - can come from multiple sources in various formats

- examples:
  - business analysing data from:
    - relational databases (structured)
    - emails (unstructured)
    - json logs (semi-structured)
  
  - Healthcare systems collecting data from:
    - electronic medical records
    - wearable heath devices
    - patient feedback forms
